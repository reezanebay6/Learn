                                                                                                                                                                Valry Ambroise Malay Ladu a year ago

                                                                                                                                                                Hello Malay Ladu,

                                                                                                                                                                You could bypass the module config file and use Finder in the service provider to scan the Modules folder:

                                                                                                                                                                $finder = Symfony\Component\Finder\Finder::create();
                                                                                                                                                                $modules = $finder->directories()->in($module_path)->depth('< 1');

                                                                                                                                                                foreach($modules as $module_object){
                                                                                                                                                                if(is_object($module_object) && $module_object->isDir()){
                                                                                                                                                                $module = $module_object->getFilename();

                                                                                                                                                                // Load the routes for each of the modules
                                                                                                                                                                if(file_exists(__DIR__.'/'.$module.'/routes.php')) {
                                                                                                                                                                require_once __DIR__.'/'.$module.'/routes.php';
                                                                                                                                                                }

                                                                                                                                                                // Load the views
                                                                                                                                                                if(is_dir(__DIR__.'/'.$module.'/Views')) {
                                                                                                                                                                $this->loadViewsFrom(__DIR__.'/'.$module.'/Views', $module);
                                                                                                                                                                }
                                                                                                                                                                }
                                                                                                                                                                }

                                                                                                                                                                All the modules added in the Modules folder will be automatically registered.

                                                                                                                                                                Hope this helps.

                                                                                                                                                                          Kamran Ahmed Mod Roman Navratil a year ago

                                                                                                                                                                          Hey, the reason why you are getting the error is because you haven't specified the namespace for the `ModuleAController`. Change the redirect line to the following and it will work:

                                                                                                                                                                          return redirect()->action('App\Modules\ModuleA\Controllers\ModuleAController@index', $id);

                                                                                                                                                                          Also note that the namespace that you have specified in the Route::group is for the routes only, you will have to specify the full controller path with the namespace in any other stuff like this redirect stub that you have

                                                                                                                                                                          return redirect()->action('App\Modules\ModuleA\Controllers\ModuleAController@index', $id);
