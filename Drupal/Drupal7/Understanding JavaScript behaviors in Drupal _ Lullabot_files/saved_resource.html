                                          Juampy NR Mod Joanna • a year ago

                                          Hi @Joana!

                                          That code looks fine to me. However, what's jq191?

                                          If the issue is with caching, can you check that the code is still being loaded into the page after reloading the window? If it is not there, then the problem will be on the server side.

                                          If you see the code being loaded into the page after refreshing, try setting a breakpoint at `$(".panorama").panorama_viewer({ repeat: false }); ` to check it is executed or not. If it's not, then set a breakpoint at `Drupal.attachBehaviors(document, Drupal.settings);`.
